# Локальное развертывание сервиса

# Инструкция по запуску сервиса
1) Скачать docker
2) Перейти в папку new_server/ . Для windows можно просто запустить run.bat двойным нажатием. Либо прописать в терминале 
```
docker compose up --build -d
```
3) В зависимости от версии проверить работоспособность сервиса можно проверить перейдя по:
	1) http://localhost:8000/ для базовой версии
	2) https://localhost/ для полной версии

# Наполнение БД
1) Положить все файлы с орбитами в папку new_server/DB_calc/computation_funcs_and_files/computation_files
2) Запустить сервис по инструкции выше
3) Зайти внутрь контейнера backend:
```
docker exec -it backend /bin/bash
```
4) Воспользоваться консольным приложением
```
python CLI.py --help # для получения информации о всех функциях приложения

python CLI.py load-family тэг_семейства #загрузить в БД соответствующее семейство
```
